Twemproxy(nutcracker)容易误解的参数

2015-08-27

Twitter，世界最大的Redis集群之一部署在Twitter用于为用户提供时间轴数据。Twitter Open Source部门提供了Twemproxy(nutracker)。

Twemproxy是一个快速的单线程代理程序，支持Memcached ASCII协议和更新的Redis协议：
它全部用C写成，使用Apache 2.0 License授权。项目在Linux上可以工作，而在OSX上无法编译，因为它依赖了epoll API.
Twemproxy 通过引入一个代理层，可以将其后端的多台 Redis 或 Memcached 实例进行统一管理与分配，使应用程序只需要在 Twemproxy 上进行操作，而不用关心后面具体有多少个真实的 Redis 或 Memcached 存储。

Java AES加密的使用及简单原理

2015-05-11

通常客户端与服务器之间的通信数据是在网络上进行传输的，因此需要对它进行加密以保证安全。最近在一次实际开发中发现，服务器AES加密后的密文到iOS客户端后无法进行解密，于是深入google了一下AES加密。发现AES加密在跨平台跨语言的地方简直是个大坑！

AES(Advanced Encryption Standard)，高级加密标准，是美国政府用于替换DES的一种加密算法标准，Java SDK中包含了部分AES的实现，注意，是“部分”实现。其实几乎每种语言对AES都是“部分”实现，这也就导致了跨语言时产生的各种坑。

登北高峰

2015-03-07

雾霾之下，共同呼吸。

北高峰海拔300多米，在杭州灵隐寺后，石磴数百级，曲折三十六湾。上有华光庙，以祀五圣。山半有马明王庙，春日祈蚕者咸往焉。峰顶浮屠七级，唐天宝中建，会昌中毁；钱武肃王修复之，宋咸淳七年复毁。

从过完年来，一直都是阴雨天气，上周日终于放晴，春游好时机。爬完回来结果就一直忙到现在也没时间，趁周末记录一下。上周末最火的事莫过于柴静的《穹顶之下》，似乎所有人一下子都觉的雾霾好恐怖，环保好重要，大有一种再不治理雾霾，马上就不能呼吸了的势头。没过几天，又开始跳出来各种各样的人，揣测柴静的各种各样的动机等等。看起来还都好像挺有理，总之在这个互联网的时代，任何事情被质疑都没什么好奇怪的。无论怎么撕逼或者谁对谁错，都要好过全世界只有一种声音。在我看来，发展要保证，环保要治理，政府要有所作为，过程要循序渐进。至于怎么实现，那就是政府公仆们该想的事了，在其位，司其职。
